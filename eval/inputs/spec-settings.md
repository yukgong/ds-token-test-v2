## 페이지 스펙: 계정 설정

### 레이아웃
왼쪽 설정 메뉴 (고정 200px) + 오른쪽 콘텐츠 영역 (스크롤 가능), 전체 100vh.
오른쪽 콘텐츠는 카드들을 세로로 쌓은 구조.

### 설정 메뉴 (왼쪽)
- 배경: 가장 연한 중립 계열
- 메뉴 상단에 뒤로가기 링크 "← 대시보드" (낮은 명도 중립 텍스트)
- 제목 "설정" (강한 중립 텍스트)
- 메뉴 항목 5개 (각 항목에 lucide 아이콘 포함):
  - 프로필 → User 아이콘 (현재 활성 상태 — 배경 강조 + 브랜드 계열 텍스트 + 브랜드 계열 아이콘)
  - 구독 플랜 → CreditCard 아이콘
  - 보안 → ShieldCheck 아이콘
  - 알림 → Bell 아이콘
  - 위험 구역 → AlertTriangle 아이콘 (되돌릴 수 없는 파괴적 액션을 나타내는 텍스트 색상 + 아이콘 색상)
- 비활성 메뉴: 낮은 명도 텍스트 + 아이콘

### 오른쪽 콘텐츠 영역

배경: 가장 연한 중립 계열.
내부 여백 있음. 카드들 사이 간격 있음.

브레드크럼 "홈 / 설정 / 프로필":
- "홈", "설정"은 클릭 가능한 링크 (낮은 명도 중립 + hover 시 브랜드 텍스트)
- "프로필"은 현재 위치 (강한 텍스트, 링크 아님)
- 구분자 "/"는 낮은 명도

---

#### 카드 1: 프로필 정보
흰색/밝은 배경 카드. 제목 "프로필 정보". 하단 저장 버튼 영역 위에 구분선.

아바타 영역 (카드 상단):
- 원형 아바타 (대형, w-20): 브랜드 계열 채움 배경 + 대비되는 텍스트 이니셜 "K"
- 아래 버튼 2개:
  - "사진 변경" (outline 스타일, 중립 테두리 + 중립 텍스트)
  - "삭제" (ghost 스타일, 낮은 명도 중립 텍스트)

입력 필드 영역 (2열 그리드):
- 이름 (label "이름", 기본 상태): 중립 테두리 + 중립/밝은 배경, placeholder 낮은 명도
- 이메일 (label "이메일", 기본 상태): 중립 테두리 + 중립/밝은 배경
- 전화번호 (label "전화번호", 비활성화 상태): 조작 불가를 나타내는 색상 처리 (배경 어둡게 + 텍스트 낮은 명도)
- 부서 (label "부서", 기본 상태): 중립 테두리 + 중립/밝은 배경

소개 textarea (label "소개", 기본 상태, 전체 너비):
- 중립 테두리 + 중립/밝은 배경
- helper text "최대 200자" (낮은 명도 중립 텍스트)

하단 버튼 영역:
- "변경사항 저장" (filled, 브랜드 계열 채움 + 대비 텍스트)
- "취소" (ghost, 낮은 명도 중립 텍스트)

---

#### 카드 2: 구독 플랜
흰색/밝은 배경 카드. 제목 "구독 플랜". (브랜드 계열 토큰 집중)

현재 플랜 박스:
- 배경: 브랜드 계열의 가장 연한 배경
- 테두리: 브랜드 계열의 연한 테두리
- 왼쪽: Zap 아이콘 (브랜드 계열 아이콘 색상) + 플랜명 "Pro" (브랜드 계열 텍스트, 굵게) + 갱신일 "2025년 9월 1일 갱신" (낮은 명도 중립 텍스트)
- 오른쪽: "현재 플랜" 뱃지 (브랜드 계열 채움 배경 + 대비되는 텍스트)

일반 안내를 나타내는 색상 계열 배너 (현재 플랜 박스 아래):
- Info 아이콘 + "결제일이 7일 후입니다. 카드 정보를 확인해 주세요." 텍스트
- 배경색 + 아이콘 색상 + 텍스트 색상 + 테두리 색상 모두 일반 안내 계열로

하단 버튼 2개:
- "Enterprise로 업그레이드" (filled, 브랜드 계열)
- "플랜 비교 보기" (outline, 중립 계열)

---

#### 카드 3: 입력 상태 쇼케이스
흰색/밝은 배경 카드. 제목 "비밀번호 변경". (입력 상태 집중 — 5가지 상태)

라벨과 함께 입력 필드 5개를 세로로 나열:

1. 현재 비밀번호 (label "현재 비밀번호", 기본 상태):
   - 중립 테두리 + 중립/밝은 배경

2. 새 비밀번호 (label "새 비밀번호", 포커스 상태):
   - 브랜드 계열 테두리 + 포커스 링 (브랜드 계열 outline, 1~2px)
   - placeholder 낮은 명도

3. 비밀번호 확인 (label "비밀번호 확인", 오류 상태):
   - 심각한 문제를 나타내는 계열 테두리 + 같은 계열 연한 배경
   - 입력값 "abc" (잘못된 입력)
   - 아래 오류 메시지: XCircle 아이콘 + "비밀번호가 일치하지 않습니다." (심각한 계열 텍스트 + 아이콘)

4. 새 이메일 (label "새 이메일 (선택)", 성공 상태):
   - 정상 완료를 나타내는 계열 테두리 + 같은 계열 연한 배경
   - 입력값 "user@example.com"
   - 아래 성공 메시지: CheckCircle2 아이콘 + "사용 가능한 이메일입니다." (완료 계열 텍스트 + 아이콘)

5. 사용자 ID (label "사용자 ID", 비활성화 상태):
   - 조작 불가를 나타내는 색상 처리 (배경 어둡게 + 텍스트 낮은 명도 + 테두리 연하게)
   - 입력값 "user_12847" (읽기 전용)
   - helper text "변경 불가" (낮은 명도)

하단 저장 버튼: filled, 브랜드 계열.

#### 카드 4: 보안
흰색/밝은 배경 카드. 제목 "보안 설정".

2단계 인증 토글 행:
- 왼쪽: ShieldCheck 아이콘 (완료/정상 계열) + "2단계 인증" 제목 + "계정 보안이 강화됩니다" 설명 (낮은 명도)
- 오른쪽: 켜진 상태 토글 (브랜드 계열 배경 + 흰 원형 핸들)
- 오른쪽 아래: "활성화됨" 뱃지 (완료/정상 계열 연한 배경 + 테두리 + 텍스트)

구분선.

활성 세션 목록 (제목 "활성 세션"):
- 세션 카드 1 (현재 세션):
  - 배경: 브랜드 계열의 가장 연한 배경 / 테두리: 브랜드 계열 연한 테두리
  - Monitor 아이콘 (브랜드 계열) + "Chrome · macOS" + "현재 위치 · 방금 전"
  - 오른쪽: "현재" 뱃지 (브랜드 계열 채움 + 대비 텍스트)
- 세션 카드 2 (다른 세션):
  - 배경: 중립 연한 배경 / 테두리: 중립 테두리
  - Smartphone 아이콘 (중립 아이콘) + "Safari · iPhone" + "서울 · 2시간 전"
  - 오른쪽: "종료" 버튼 (outline, 심각한 계열 테두리 + 심각한 계열 텍스트)
- 세션 카드 3 (다른 세션):
  - 같은 스타일 / "Firefox · Windows" + "부산 · 3일 전"
  - 오른쪽: "종료" 버튼 (같은 스타일)
- 하단: "모든 다른 세션 종료" 버튼 (outline, 심각한 계열)

---

#### 카드 5: 알림 설정
흰색/밝은 배경 카드. 제목 "이메일 알림".

주의가 필요한 상태를 나타내는 계열 배너 (카드 상단):
- AlertTriangle 아이콘 + "일부 알림을 끄면 중요 서비스 공지를 놓칠 수 있습니다."
- 배경색 + 테두리 + 아이콘 + 텍스트 모두 주의 계열로

알림 항목 목록 (체크박스 + 라벨 + 설명 텍스트):

체크박스 스타일:
- 체크된 상태: 브랜드 계열 채움 배경 + 대비 텍스트(체크마크)
- 체크 안된 상태: 중립 테두리 + 중립/밝은 배경
- 비활성화 + 체크된 상태: 조작 불가 색상 처리 (명도 낮게)

항목 6개:
1. 시스템 알림 — 체크됨 + 비활성화 처리 (필수 항목, 변경 불가) + helper "서비스 이용에 필요한 필수 공지"
2. 결제 알림 — 체크됨 + helper "영수증 및 결제 관련 안내"
3. 배포 알림 — 체크됨 + helper "배포 완료 및 실패 알림"
4. 보안 알림 — 체크됨 + helper "로그인 및 보안 이벤트"
5. 마케팅 알림 — 체크 안됨 + helper "제품 업데이트 및 프로모션"
6. 뉴스레터 — 체크 안됨 + helper "월간 뉴스레터"

하단 저장 버튼: filled, 브랜드 계열.

---

#### 카드 6: 위험 구역
- 배경: 심각한 문제를 나타내는 계열의 가장 연한 배경
- 테두리: 심각한 계열 연한 테두리
- 카드 헤더: AlertTriangle 아이콘 (심각한 계열) + "위험 구역" 제목 (심각한 계열 텍스트)

위험 액션 2개 (각각 구분선으로 분리):

액션 1 — 계정 비활성화:
- 제목 "계정 비활성화" + 설명 "계정을 일시적으로 비활성화합니다. 언제든 재활성화할 수 있습니다." (낮은 명도 중립)
- 버튼: "비활성화" (outline 스타일, 심각한 계열 테두리 + 심각한 계열 텍스트)

액션 2 — 계정 영구 삭제:
- 제목 "계정 영구 삭제" + 설명 "계정과 모든 데이터가 영구 삭제됩니다. 되돌릴 수 없습니다." (낮은 명도 중립)
- 버튼: "영구 삭제" (filled 스타일, 심각한 계열 채움 배경 + 대비 텍스트)
