## 페이지 스펙: 결제 및 구독 관리

### 레이아웃
왼쪽 사이드바 (고정 240px) + 오른쪽 메인 콘텐츠 (스크롤), 전체 100vh.
사이드바는 대시보드와 동일한 어두운 solid 계열 스타일로 구성.

### 사이드바
- 배경: 어두운 solid 계열
- 상단: 로고 아이콘 (LayoutDashboard) + "Mildang" 텍스트
- 네비 메뉴: 결제 내역 (CreditCard, 활성 상태, 어두운 배경 + 밝은 텍스트) / 대시보드 / 사용자 관리 / 설정 (비활성 (opacity: 0.38))
- 하단: "로그아웃" + LogOut 아이콘 (파괴적 액션 계열 색상)

### 메인 콘텐츠
배경: 가장 연한 중립 계열. 내부 여백. 섹션들 세로로 나열.

#### 섹션 1: 플랜 선택 (3열 카드)

카드 공통: 흰색/밝은 배경, 테두리, rounded-xl, 내부 여백.

**Free 카드 (왼쪽)**:
- 카드 배경: 기본 흰색/밝은 배경 + 중립 테두리
- 플랜명 "Free" (강한 중립 텍스트)
- 가격 "₩0/월" (큰 텍스트, 강한 중립)
- 기능 목록 5개:
  - 포함 기능 3개: Check 아이콘 (완료 계열 아이콘 색상) + 기능명 (중립 텍스트)
  - 제외 기능 2개: X 아이콘 (비활성 (opacity: 0.38)) + 기능명 (비활성 (opacity: 0.38))
- 하단 버튼: "현재 플랜" (비활성 (opacity: 0.38))

**Pro 카드 (중앙, 추천/현재)**:
- 카드 상단 배너 (rounded-t-xl): 메인 컬러 계열 채움 배경 + 대비 텍스트 "추천"
- 카드 배경: 메인 컬러 계열의 가장 연한 배경
- 카드 테두리: 메인 컬러 계열 (기본보다 강한 단계)
- 플랜명 "Pro" (메인 컬러 계열 텍스트, 굵게)
- "현재 플랜" 뱃지 (메인 컬러 계열 채움 + 대비 텍스트) — 플랜명 옆
- 가격 "₩49,000/월" (큰 텍스트, 메인 컬러 계열 텍스트)
- 기능 목록 6개:
  - 포함 기능 5개: Check 아이콘 (메인 컬러 계열 아이콘) + 기능명 (메인 컬러 계열 텍스트 또는 강한 중립)
  - 제외 기능 1개: X 아이콘 (비활성 (opacity: 0.38)) + 기능명 (비활성 (opacity: 0.38))
- 하단 버튼: "현재 사용 중" (비활성 (opacity: 0.38), 메인 컬러 계열 채움)
- 카드 하단 메모: "다음 결제일: 2025년 9월 1일" (비활성 (opacity: 0.38))

**Enterprise 카드 (오른쪽)**:
- 카드 배경: 기본 흰색/밝은 배경 + 중립 테두리
- 플랜명 "Enterprise" (강한 중립 텍스트)
- 가격 "맞춤 견적" (큰 텍스트, 강한 중립)
- 기능 목록 6개: 전부 포함 (Check 아이콘, 완료 계열)
- 하단 버튼: "영업팀 문의" (메인 컬러 계열 채움 + 대비 텍스트)

#### 섹션 2: 결제 수단

흰색/밝은 배경 카드. 제목 "등록된 결제 수단".

결제 카드 표시 (가로 배치):

  - 상단: CreditCard 아이콘 (비활성 (opacity: 0.38))
  - 하단: "•••• •••• •••• 4242" (강한 텍스트)
- 카드 정보:
  - 카드사 "Visa" (강한 텍스트)
  - 만료일 "12/26" (중간 명도) + "유효" 뱃지 (완료 계열 연한 배경 + 테두리 + 텍스트)
