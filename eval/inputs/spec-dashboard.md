## 페이지 스펙: 서비스 모니터링 대시보드

### 레이아웃
왼쪽 사이드바 (고정 240px) + 오른쪽 메인 콘텐츠, 전체 100vh.

### 사이드바
- 배경: 어두운 solid 계열
- 상단: 로고 아이콘 + "Mildang" 텍스트
- 네비 메뉴 5개 (각 항목에 lucide 아이콘 포함):
  - 대시보드 → LayoutDashboard 아이콘 (현재 활성 상태 — 어두운 배경 + 밝은 텍스트)
  - 사용자 관리 → Users 아이콘
  - 결제 내역 → CreditCard 아이콘
  - 알림 설정 → Bell 아이콘
  - 시스템 로그 → ScrollText 아이콘
  - 비활성 메뉴: 비활성 (opacity: 0.38)
- 하단: "로그아웃" 텍스트 + LogOut 아이콘 (되돌릴 수 없는 파괴적 액션을 나타내는 색상)

### 메인 콘텐츠
배경: 가장 연한 중립 계열

#### 상단 헤더 바
- 밝은 배경, 하단 테두리
- 왼쪽: 브레드크럼 "홈 / 모니터링 / 대시보드"
  - "홈", "모니터링"은 클릭 가능한 링크 (기본 색상 + hover 시 색상 변화)
  - "대시보드"는 현재 위치 (강조된 텍스트, 링크 아님)
  - 구분자 "/" 는 비활성 (opacity: 0.38)
- 오른쪽:
  - 아이콘 버튼 3개 (ghost 스타일, hover 시 배경 변화):
    - Search 아이콘 버튼 — hover 시 툴팁 "검색" 표시 (어두운 배경 + 대비 텍스트)
    - Bell 아이콘 버튼 + 문제 발생을 나타내는 색상의 숫자 뱃지 (3) — 툴팁 "알림"
    - Settings 아이콘 버튼 — 툴팁 "설정"
  - 세로 구분선
  - 메인 컬러 주요 색상으로 채워진 filled 버튼 "새 배포" (Plus 아이콘 포함, hover 시 색상 변화)
  - 프로필 아바타: 원형, 메인 컬러 계열 배경 + 대비되는 텍스트 색상으로 이니셜 "K"

툴팁 스타일: 어두운 solid 배경 + 대비되는 텍스트 색상 (일반 콘텐츠 영역과 반전된 느낌).

#### 토스트 알림 영역 (헤더 바 바로 아래, 우측 정렬)
토스트 3개를 세로로 쌓아 표시 (정적 렌더링):
1. 정상 완료를 나타내는 색상 계열 / CheckCircle2 아이콘 / X 닫기 버튼 / "배포 v2.4.1이 완료되었습니다."
2. 심각한 문제를 나타내는 색상 계열 / XCircle 아이콘 / X 닫기 버튼 / "DB 연결이 끊어졌습니다. 즉시 확인하세요."
3. 중립적인 일반 안내를 나타내는 색상 계열 / Info 아이콘 / X 닫기 버튼 / "시스템 점검이 2025-06-10 02:00에 예정되어 있습니다."

각 토스트: 배경색 + 아이콘 색상 + 텍스트 색상 + 테두리 색상 모두 해당 의미의 시맨틱 색상 계열로.

#### 통계 카드 4개 (2×2 그리드)
흰색/밝은 배경, 테두리 있음. 카드 우상단에 lucide 아이콘.
1. 총 사용자 12,847 / +12% — 좋은 변화, TrendingUp 아이콘
2. 결제 실패 24건 / +3건 — 문제 상황, AlertCircle 아이콘
3. 평균 응답시간 142ms / -8% — 좋은 변화, Zap 아이콘
4. 미처리 민원 7건 / +2건 — 주의 필요, Clock 아이콘

각 카드 하단의 변화량 뱃지: 배경색 + 텍스트색 + 테두리색 모두 해당 의미의 시맨틱 색상 계열.
아이콘 색상도 같은 시맨틱 색상 계열로.

#### 시스템 리소스 (카드 형태, 가로 3개)
각 카드: 라벨 + 수치 + 프로그레스 바 + 상태 텍스트.
1. CPU 사용량 42% — 여유로운 상태를 나타내는 색상으로 채워진 바
2. 메모리 사용량 78% — 주의가 필요한 상태를 나타내는 색상으로 채워진 바
3. 디스크 사용량 91% — 심각한 문제를 나타내는 색상으로 채워진 바

프로그레스 바: 채워진 부분은 해당 의미의 시맨틱 색상, 빈 부분은 중립 계열 배경.

#### 최근 이벤트 테이블
테이블 헤더 영역:
- 왼쪽: 탭 3개 ("전체 이벤트" 활성 / "미처리" / "처리완료")
  - 활성 탭: 하단 강조선 + 강한 텍스트 색상
  - 비활성 탭: 비활성 (opacity: 0.38), hover 시 중간 명도로 변화
- 오른쪽 버튼 2개:
  - "필터" (outline 스타일, Filter 아이콘)
  - "CSV 내보내기" (비활성 (opacity: 0.38), Download 아이콘)

탭 아래 태그 필터 바 (가로로 나열):
- "전체" (선택됨 — 메인 컬러 배경 + 대비 텍스트)
- "로그인" (미선택 — 중립 배경 + 중립 텍스트, hover 시 변화)
- "결제" (미선택)
- "보안" (미선택)
- "시스템" (미선택)

검색 인풋 (태그 필터 바 오른쪽):
- 기본 상태: 중립 테두리 + 중립 배경
- placeholder 텍스트: 비활성 (opacity: 0.38)
- 포커스 상태: 메인 컬러 계열 테두리 + 포커스 링
- (아래에 오류 상태 인풋을 별도로 하나 더 보여줌: 잘못된 날짜 형식 입력 — 문제를 나타내는 색상의 테두리 + 배경 + 오류 메시지 텍스트)

컬럼: 사용자 / 이벤트 / 시간 / 상태 / 액션
행 5개 (사용자 열에 아바타 + 이름 나란히):
- 아바타(이니셜 "홍") / 로그인 성공 / 14:32 / 정상 처리됨 / 아이콘 버튼 (Eye)
- 아바타(이니셜 "김") / 결제 실패 / 14:28 / 문제 발생 / 아이콘 버튼 (Eye)
- 아바타(이니셜 "이") / 비밀번호 5회 오류 / 14:21 / 주의 필요 / 아이콘 버튼 (Eye)
- 아바타(이니셜 "박") / 회원가입 / 14:15 / 정상 처리됨 / 아이콘 버튼 (Eye)
- 아바타(이니셜 "시") / API 응답 지연 / 14:09 / 주의 필요 / 아이콘 버튼 (Eye)

아바타: 원형, 중립 계열 배경 + 대비 텍스트 색상.
상태 열: 아이콘 + 텍스트, 색상은 해당 의미의 시맨틱 색상 계열.
아이콘 버튼 (Eye): ghost 스타일, hover 시 배경색 변화.
짝수 행 배경: 홀수 행과 구분되는 줄무늬 효과.

#### 알림 설정 토글 패널 (테이블 아래, 카드 형태)
제목 "알림 수신 설정" + 토글 스위치 4개:
1. 배포 알림 — 켜진 상태 (메인 컬러 계열 배경으로 채워진 토글)
2. 장애 알림 — 켜진 상태
3. 보안 알림 — 꺼진 상태 (중립 계열 배경의 토글)
4. 마케팅 알림 — 꺼진 상태 (비활성 (opacity: 0.38))

토글 켜진 상태: 메인 컬러 계열 배경 + 흰 원형 핸들.
토글 꺼진 상태: 중립 계열 배경 + 흰 원형 핸들.
비활성화 토글: 비활성 (opacity: 0.38).

각 카드: lucide 아이콘 + 제목 + 설명 텍스트 + 닫기(X) 아이콘 버튼 + 하단 액션 버튼.
1. 정상 완료를 나타내는 색상 계열 / CheckCircle2 / "배포 완료" / "v2.4.1이 성공적으로 배포되었습니다."

   → 하단 버튼: "즉시 확인" (해당 색상 계열의 filled 버튼) + "무시" (ghost 버튼)

   → 하단 버튼: "정리 시작" (해당 색상 계열의 outline 버튼)
